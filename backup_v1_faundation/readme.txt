============================================
 1. バックアップに含まれるファイル（必須データ）
============================================
[main.py]
  - システムの頭脳となるPythonプログラム。
  - テキストを監視し、JSONから回答を選び、OBSへの指示ファイルを作成する。
  - F9キー監視機能付き。

[auto_wav_player.lua]
  - OBS制御用のスクリプト。OBSに読み込ませて使用。
  - テキストファイルの変更を検知し、メディアソースの再生やソースの表示/非表示を行う。

[microwave_data.json]
  - ゲームのルールデータ（今回は電子レンジ用）。
  - 質問キーワードと再生する音声ファイルの対応表。

[audioフォルダ]
  - 回答用の音声素材（WAVファイル）一式。

============================================
 2. 実行時に親フォルダに必要なファイル（一時ファイル）
 ※バックアップには含めていないが、実行には必須。
   なければ空のテキストファイルとして新規作成すること。
============================================
[current_question.txt]
  - 役割：【耳】（入力）
  - 説明：LocalVocal(Whisper)が、聞き取った質問テキストを書き込むファイル。
  - Pythonがこのファイルの更新を監視している。

[next_wav_path.txt]
  - 役割：【口】（音声指示）
  - 説明：Pythonが「次に再生するWAVファイルのフルパス」を書き込むファイル。
  - OBS(Lua)がこれを読み取り、メディアソースを再生する。

[thinking_state.txt]
  - 役割：【表情・状態】（演出指示）
  - 説明：現在の状態を数字で管理するファイル。
  - 0 = 待機中（マイクON / 文字OFF）
  - 1 = 考え中（マイクOFF / 文字ON）
  - 2 = 一時停止（マイクOFF / 文字OFF）
